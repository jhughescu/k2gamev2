About Access Keys

Access keys (AK) grant password-protected access to the Facilitator Dashboard, where facilitators can access the entry point for their instance of the K2 game and view/download the data produced.

Institution keys grant access across all courses in the selected institution.

Course keys limit access to one specific course.

End date is optional and limits the time period during which the game can be played for that AK.

Session limit is optional and caps how many sessions can be created for that AK.
